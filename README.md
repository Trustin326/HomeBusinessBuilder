# Home Business Builder — Microsoft Partner‑Style AI Suite (Front‑End Demo)

This is a **one‑file deployable** (static) demo you can host on **GitHub Pages**. It’s inspired by Microsoft partner aesthetics but is **not affiliated with Microsoft**.

## How to deploy on GitHub Pages (no terminal)
1. Create a new repo (e.g., `HomeBusinessBuilder`).
2. Upload the entire contents of this ZIP (keep folders intact).
3. Go to **Settings → Pages → Branch: main / root** → Save.
4. Open the published URL (e.g., `https://username.github.io/HomeBusinessBuilder/`).

## Features (with live demos)
- Doc Maker (Word‑like) — export .txt
- Email Tracker — simulate opens, CSV export (localStorage)
- Slide Generator — outline to slide cards
- Spreadsheet Lite — basic totals with `=SUM`
- Blog Assistant — title → outline → draft text
- Sales Copy Assistant — PAS + bullets + CTA
- Website Generator — export a single‑file landing page
- Leads Tracker — form to table, CSV export

## Customize
- Replace the **Apply/Buy** form with your Stripe/WarriorPlus link.
- Colors: edit `assets/css/style.css`.
- Logic: edit `assets/js/app.js`.
